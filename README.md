### Tech Stack
- python
- flask
- jinja templates
- AWS S3
- pillow library
- HTML
- CSS
- bootstrap

### Run on your machine

```
git clone https://github.com/brendaliu2/Pixly.git
flask run -p 5001
```

### Enviornment Variables
* ACCESS_KEY
* SECRET_ACCESS_KEY
* BUCKET
* BASE_URL
* DATABASE_URL
